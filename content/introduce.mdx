# Project Introduction

Welcome to **learn-next-16**, a modern full-stack web application built with cutting-edge technologies and best practices for scalable, production-ready development.

## Technology Stack

### Frontend Framework
- **Next.js 16** (Canary) - React framework with App Router, Server Components, and experimental Partial Prerendering (PPR)
- **React 19** - Latest React version with enhanced server components and concurrent features
- **TypeScript 5** - Type-safe development with full type checking

### Authentication & Authorization
- **Better Auth v1.3.34** - Modern authentication library with built-in support for:
  - Email and password authentication
  - Admin plugin for user management
  - Access control plugin for role-based permissions
  - Secure session management with cookie-based authentication

### Database
- **PostgreSQL** - Robust relational database for data persistence
- **node-postgres (pg)** - Native PostgreSQL driver for Node.js

### UI/UX
- **Tailwind CSS v4** - Utility-first CSS framework for rapid UI development
- **Tailwind Typography** - Beautiful typographic defaults for content-rich pages
- **shadcn/ui** - High-quality, accessible React components built on Radix UI:
  - Avatar, Button, Checkbox, Dropdown Menu, Input, Label, Separator, Skeleton
- **Lucide React** - Modern icon library with 1000+ carefully crafted icons
- **next-themes** - Perfect dark mode implementation with zero-flicker theme switching
- **Sonner** - Elegant toast notifications

### MDX Support
- **@mdx-js/react & @mdx-js/loader** - Write JSX in markdown documents
- **@next/mdx** - First-class MDX support in Next.js with full component integration

### Form Validation
- **Zod v4** - TypeScript-first schema validation for forms and API endpoints

### Development Tools
- **ESLint 9** - Code quality and consistency enforcement
- **next-config with turbopack** - Ultra-fast bundler for development

## Development Workflow

1. **Local Development**: Run `npm run dev` for hot-reload development server on port 3002
2. **Type Checking**: TypeScript validates code at build time and in IDE
3. **Linting**: ESLint ensures code quality and consistency
4. **Building**: `npm run build` creates optimized production bundle with standalone output
5. **Production**: `npm run start` serves the built application

## Security Features

- Environment variable management with `.env.local`
- CSRF protection built into Better Auth
- Secure cookie attributes (HttpOnly, Secure, SameSite)
- SQL injection prevention through parameterized queries
- Type-safe database operations with Zod validation

## Scalability Considerations

- **Standalone output** - Minimal footprint for containerization
- **Database pooling** - Efficient PostgreSQL connection management
- **Server Components** - Reduced client-side JavaScript
- **Incremental Static Regeneration** - Balance between static and dynamic content
- **API route optimization** - Efficient data fetching and caching strategies
